# Git_Lab

* Helloworld.java是一个简单的Java程序，用于输出三次"Hello World!"。

   ```java

    public class Helloworld {
        public static void main(String[] args) {
            System.out.println("Hello World!");
            System.out.println("Hello World!");
            System.out.println("Hello World!");
        }
    }
   ```
